Task 3 – 10 Types of Web Attacks

Name Web Attacks
SQL Injection
SQL Injection exploits vulnerabilities in database-driven applications. Attackers insert malicious SQL statements into input fields. This can result in unauthorized access or data loss.
Reference – OWASP
Cross-Site Scripting (XSS)
XSS allows attackers to inject malicious scripts into web pages. These scripts execute in users' browsers and can steal session data.
Reference – OWASP
Cross-Site Request Forgery (CSRF)
CSRF tricks authenticated users into submitting unintended requests. It exploits trust between browser and website.
Reference – OWASP
Phishing
Phishing deceives users into revealing personal information. Attackers impersonate trusted organizations through fake websites or emails.
Reference – ACSC
Man-in-the-Middle (MITM)
MITM intercepts communication between two parties. Attackers can eavesdrop or modify transmitted data.
Reference – OWASP
DDoS
DDoS floods servers with massive traffic making services unavailable to legitimate users.
Reference – Cloudflare
Brute Force Attack
Brute force attacks attempt many password combinations until the correct one is found.
Reference – OWASP
Session Hijacking
Session hijacking steals session IDs allowing attackers to impersonate users.
Reference – OWASP
Clickjacking
Clickjacking tricks users into clicking hidden elements performing unintended actions.
Reference – OWASP
Malware Injection
Malware injection inserts malicious code into websites compromising visitors' systems.
Reference – CISA
